# easiness

## Installation

    $ npm install easiness

## Require

```javascript
var easiness = require('easiness')
```

## API

#### `easiness.$easeFn(x)`

Replace `$easeFn` with one of:

    * `linear`
    * `smooth`
    * `easeInQuad`
    * `easeOutQuad`
    * `easeInOutQuad`
    * `easeInSine`
    * `easeOutSine`
    * `easeInOutSine`

#### `easiness.$easeFn.mapped(inStart, inEnd, outStart, outEnd)`
